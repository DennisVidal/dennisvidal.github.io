Noteworthy Projects

Research Ravine

A flight simulation that was created as part of my master thesis. It utilises isosurface extraction and procedural generation in combination with compute shaders, VR and an Icaros system to deliver an immersive and unique experience with every play session.

Serious Game - Compulsive Hoarding

A VR serious game that is designed to be used in the therapie of compulsive hoarding. It is the result of a collaboration of seven students in a small scale group project.

  • PROJECT PAGE

Nonverbal Communication

An application that is the result of my bachelor thesis. It allows for the interaction with a character through a row of nonverbal means, such as eye contact and distance to tthe character. The game makes use of AR via a HoloLens to embed the character in the real world.

  • PROJECT PAGE

DayZ Mod - Banking

A mod for DayZ which adds a banking system to the game. It allows server owners to configure and place ATMs throughout the world, which players can use to deposit and withdraw money.

  • GITHUB
  • STEAM

DayZ Mod - Fast Travel

A mod for DayZ which adds a fast travel system to the game. It allows server owners to configure and place fast travel boards in the world. Players can use these boards to teleport between predefined locations.
